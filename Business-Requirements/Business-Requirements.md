# Executive Summary

Cyberjay Enterprise WAN is a multi-branch enterprise network designed to connect the Nairobi Headquarters, Mombasa Branch, and Kisumu Branch through a secure and scalable Wide Area Network (WAN).

The network will provide centralized authentication, file sharing, network management, and secure communication between all locations.

---

# Company Background

Cyberjay Enterprise is a technology company providing:

* Network Infrastructure Solutions
* Cybersecurity Services
* IT Support and Maintenance
* Technology Consulting

The company operates from Nairobi Headquarters with regional branches in Mombasa and Kisumu.

---

# Business Objectives

The organization requires a network that:

* Connects all branches securely.
* Centralizes management of users and devices.
* Supports future company growth.
* Minimizes downtime.
* Protects company information.
* Enables efficient collaboration between branches.

---

# Functional Requirements

The network must provide:

* User authentication.
* File sharing.
* Internet connectivity.
* Email communication.
* Centralized management.
* Secure inter-branch communication.
* Network monitoring.

---

# Non-Functional Requirements

The network must be:

* Reliable.
* Secure.
* Scalable.
* Easy to manage.
* Fault tolerant.

# Success Criteria

The project shall be considered successful when:

* All branches communicate successfully.
* Users authenticate through Active Directory.
* OSPF routing operates correctly.
* ACLs enforce security policies.
* Redundancy provides failover capability.
* Documentation is complete.
* Network services remain available during simulated failures.
